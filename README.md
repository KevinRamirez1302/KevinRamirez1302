## 💡 Proyectos Destacados

### 🃏 Mazo App — Enterprise Architecture
> **Proyecto Final Destacado (Ciclo DAM)**
> Una aplicación robusta diseñada para la gestión de mazos de cartas, centrada en la arquitectura limpia y la contenerización. Representa los estándares de calidad del ciclo de Desarrollo de Aplicaciones Multiplataforma.

- **Backend:** Desarrollado con **NestJS** y TypeScript aplicando inyección de dependencias y arquitectura modular.
- **Seguridad & Infra:** Autenticación robusta, hashing con **Bcrypt** y base de datos **PostgreSQL** orquestada completamente en **Docker**.
